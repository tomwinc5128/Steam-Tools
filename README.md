## Overview
**Steam Tools** is a powerful collection of utilities designed to enhance your Steam experience. It includes various tools for managing games, optimizing performance, customizing your library, and improving overall usability.

> [!Important]
> This project is intended for use with legitimate Steam accounts and games you legally own.

## Installation
### For Regular Users
1. Download and extract the archive
2. Run the main executable as Administrator
3. Select the desired tool from the menu
4. Enjoy enhanced Steam experience

## Troubleshooting
| Problem                    | Solution                                      |
|----------------------------|-----------------------------------------------|
| App doesn't start          | Run as Administrator                          |
| Tools not working          | Make sure Steam is running                    |
| Slow performance           | Close background applications                 |
| Antivirus warning          | Add the folder to exclusions                  |
| UI looks blurry            | Enable DPI scaling override in compatibility  |
